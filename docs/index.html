تسجيل الدخول

⚙️ كيف يعمل النظام؟

يوفر النظام مرجعاً تشغيلياً موحداً لمتابعة متطلبات الجهات التنظيمية وإدارة عناصر التشغيل المرتبطة بالفروع بشكل مستمر.

1️⃣
إدارة الفروع ومراكز التكلفة

إضافة الفروع وربطها بالعلامة التجارية والمنطقة و Cost Center ضمن لوحة مركزية.

2️⃣
إدارة العقود والتراخيص

رفع وربط المستندات (بلدية، دفاع مدني، إيجارات، تصاريح، كروت صحية، تدريب) مع متابعة تواريخ الانتهاء.

3️⃣
إدارة السكنات

متابعة بيانات السكن والعقود والتواريخ التشغيلية المرتبطة بالموظفين والفروع.

4️⃣
إدارة المواصلات

تسجيل بيانات الحافلات والمركبات وربطها بالفروع والموظفين مع متابعة الحالة التشغيلية.

5️⃣
مراقبة التشغيل ومستوى المخاطر

عرض الفروع المعرضة للمخالفة، حالة الامتثال، التنبيهات القادمة، وأسباب ارتفاع المخاطر.

⭐ وحدات النظام الرئيسية

🔔
مركز التنبيهات

متابعة جميع تواريخ الانتهاء والتنبيهات التشغيلية في شاشة واحدة.

📋
إدارة المخالفات

تسجيل وتحليل المخالفات وربطها بالفروع لتقليل التكرار.

📑
العقود والتراخيص

إدارة العقود والرخص والتصاريح مع الأرشفة وتنبيهات الانتهاء.

🏠
السكنات

متابعة السكن والطاقة الاستيعابية والارتباط بالفروع.

🚌
المواصلات

إدارة بيانات النقل والتكاليف وربطها بالفروع.

📊
مؤشر خطر الفروع

تحليل تشغيلي ذكي يدعم اتخاذ القرار.

🔐
الأمان والصلاحيات

نظام صلاحيات حسب الدور، عزل بيانات الفروع، وتخزين مرفقات آمن.

✅ الهدف من النظام: توفير رؤية تشغيلية موحدة تساعد على قرارات استباقية وتقليل المخاطر.
