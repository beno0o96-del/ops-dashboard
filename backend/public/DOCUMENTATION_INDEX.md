## 🎯 رحلة القراءة الموصى بها

### 👤 إذا كنت مستخدماً عادياً
```
README.md (5 دقائق)
  ↓
INSTALLATION_GUIDE.md (10 دقائق)
  ↓
test-api.html (5 دقائق للاختبار)
  ↓
جاهز للاستخدام! ✅
```

### 💻 إذا كنت مطوراً
```
README.md (5 دقائق)
  ↓
SUMMARY.md (15 دقيقة)
  ↓
UNIFIED_SAVE_SYSTEM.md (20 دقيقة)
  ↓
ادرس الأكواد في js/ و backend/
  ↓
جاهز للتطوير! ✅
```

### 🔧 إذا واجهت خطأ
```
error message
  ↓
TROUBLESHOOTING.md
  ↓
ابحث عن الخطأ الخاص بك
  ↓
اتبع الحل الموصى به
  ↓
إذا استمر الخطأ:
  ↓
DevTools (F12) + console.log
  ↓
UNIFIED_SAVE_SYSTEM.md للتحقق من الاستخدام
```

## 🔑 المفاهيم الرئيسية

### 🔄 API (واجهة البرمجة)
**ما هي؟**
- طريقة للتواصل بين Frontend و Backend
- تستخدم HTTP (POST, GET, PUT, DELETE)
- ترسل وتستقبل JSON

**أين تقرأها؟**
- UNIFIED_SAVE_SYSTEM.md - قسم "API للموارد"

---

### 💾 قاعدة البيانات
**ما هي؟**
- مكان تخزين البيانات الآمن
- تدعم البحث والفرز السريع
- يمكن النسخ الاحتياطي

**أين تقرأها؟**
- INSTALLATION_GUIDE.md - قسم "قاعدة البيانات"

---

### 📝 الـ Models و Controllers
**ما هي؟**
- Models = جداول قاعدة البيانات
- Controllers = وظائف API

**أين تقرأها؟**
- backend/app/Http/Controllers/
- backend/app/Models/

---

### 🎨 Frontend
**ما هو؟**
- الواجهة التي يراها المستخدم
- HTML + CSS + JavaScript

**أين تقرأه؟**
- ملفات .html في المشروع الرئيسي
- js/ للملفات الجديدة
